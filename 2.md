##  File & Directory Management

```bash
mkdir test          # Create a directory named 'test'
rmdir test          # Remove an empty directory
rm file.txt         # Delete a file
rm -r folder        # Delete a folder recursively
cp a.txt b.txt      # Copy a.txt to b.txt
mv a.txt folder/    # Move a.txt into folder/
```

---

##  Viewing & Editing Files

```bash
cat file.txt        # Display file content
less file.txt       # View file page by page
head file.txt       # Show first 10 lines
tail file.txt       # Show last 10 lines
nano file.txt       # Edit file using nano editor
vi file.txt         # Edit file using vi editor
```

---

## System Information

```bash
whoami              # Show current user
uname -a            # Show system information
df -h               # Disk space usage (human readable)
du -sh *            # Size of files/folders in current directory
free -h             # Memory usage
uptime              # Show system uptime
```

---

##  Environment Variables & Shell

```bash
env                 # Show all environment variables
echo $HOME          # Display HOME variable
export MYVAR=123    # Create/set environment variable
history             # Show command history
clear               # Clear terminal screen
```
