## enable pubsub
```bash
gcloud services enable pubsub.googleapis.com
```
---
## create pubsub topic
```bash
gcloud pubsub topics create my-topic
```
---
## Create a Subscription
```bash
gcloud pubsub subscriptions create my-subscription --topic=my-topic
```
---
## publish message
```bash
gcloud pubsub topics publish my-topic --message="Hello, Pub/Sub!"
```
---
## receive message(2nd terminal)
```bash
gcloud pubsub subscriptions pull my-subscription  --auto-ack
```
---
## limit message
```bash
gcloud pubsub subscriptions pull my-subscription --auto-ack --limit=3
```
## 2nd terminal
```bash
gcloud pubsub topics publish my-topic --message="Message 1"
gcloud pubsub topics publish my-topic --message="Message 2"
gcloud pubsub topics publish my-topic --message="Message 3"
```
---
## clean up resource 
```bash
gcloud pubsub subscriptions delete my-subscription
```
```bash
gcloud pubsub topics delete my-topic
```
